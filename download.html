動作環境

 macOS

  • macOS 11.0 (Big Sur) 以降
  • Apple Silicon (M1以降) または Intel 64-bit
  • メモリ: 4GB 以上推奨
  • ストレージ: 200MB 以上の空き容量

Windows

  • Windows 10 / 11 (64-bit)
  • x64 アーキテクチャ
  • メモリ: 4GB 以上推奨
  • ストレージ: 200MB 以上の空き容量
  • Visual C++ 再頒布可能パッケージ (自動インストール)

よくある質問

はい、CapCell は完全無料でご利用いただけます。アカウント登録も不要です。
macOS: ダウンロードした .dmg ファイルを開き、CapCell を Applications フォルダにドラッグしてください。初回起動時に「開発元を確認できない」と表示された場合は、右クリック →「開く」を選択してください。

Windows: ダウンロードした .msi または .exe ファイルを実行し、インストーラーの手順に従ってください。
すべてのプロジェクトデータはローカルに保存されます。クラウドへの送信は一切行いません。プロジェクトは .CapCell.json 形式でエクスポート・共有することができます。
事前に VS Code をインストールし、ターミナルから code コマンドが使えることを確認してください。CapCell の設定画面でプロジェクトのベースパスを設定すると、ノードから VSCode で直接ファイルを開けるようになります。
現在 macOS・Windows 版のみ提供しています。Linux 版については今後のリリースで対応予定です。GitHub でアップデートをご確認ください。
GitHub のリポジトリ から Issue を作成してお知らせください。フィードバックをお待ちしています。
